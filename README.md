# 🔐 Secure Exam Paper Distribution System

## 📌 Overview

A secure web-based system for distributing exam papers digitally with encryption and time-based access control to prevent leakage and unauthorized access.

---

## 🚀 Features

* Secure PDF upload & encryption
* Time-based paper unlocking
* Role-based access control
* Authentication & OTP verification
* Activity logging

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* SQLite

---

## ⚙️ How It Works

Admin uploads papers → system encrypts → assigns access → papers unlock at scheduled time → authorized users can download securely.
